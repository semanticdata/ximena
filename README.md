## ⛑ Useful Commands

```sh
# Install dependencies
npm install

# Update dependencies
npm update

# Start local dev server
npm start

# Check formatting w/ Prettier
npm run check

# Fix formatting w/ Prettier
npm run format
```

For more details take a look at [package.json](package.json).
